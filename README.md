# sbd_mask
sbd_mask
这是思百达物联网科技（北京）有限公司 开源的一款轻量级cpu版实时口罩检测项目。
速度和精度远超百度开源的轻量级口罩检测。

 速度是百度的两倍多左右
 精度也远超百度paddlehub的口罩检测，
